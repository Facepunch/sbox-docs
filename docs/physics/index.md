# Physics

S&box includes a full 3D and 2D physics system with great performance and stability, powered by the Source 2 engine.

Key features include rigidbodies, colliders, joints, raycasting, and trigger volumes — all usable through Components on GameObjects.

:::info
Detailed physics documentation is coming soon. In the meantime, explore the built-in physics components (Rigidbody, BoxCollider, SphereCollider, etc.) by adding them to GameObjects in the editor.
:::
