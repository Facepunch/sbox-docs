# Installation & Setup

This guide covers how to install s&box, connect it to your Steam account, and get ready to play games.

## Quick Install

To play games created in s&box, you must download the s&box client via Steam.

[Install s&box through Steam](steam://run/2129370)

1. Click the link above or search for "s&box" in your Steam Library if you already have access.
2. Click **Install**.
3. Choose your installation drive (s&box can grow large due to downloaded addons, so a drive with plenty of space is recommended).
4. Once downloaded, click **Play** to launch the Main Menu.

## First Launch

When you launch s&box for the first time, you will be greeted by the Main Menu. From there you can customize your avatar, browse games, and join servers.

## Account Setup

s&box uses your Steam account for authentication.
* Your display name in-game will match your Steam profile name.
* Your Steam ID is used to save your progress across different games and servers.
* **No extra accounts are required.**

## System Requirements

While s&box games vary wildly in their performance requirements, the base engine requires a modern PC.

| Requirement | Minimum | Recommended |
| :--- | :--- | :--- |
| **OS** | Windows 10 64-bit | Windows 11 64-bit |
| **Processor** | Intel i5 or AMD equivalent | Intel i7 or AMD equivalent |
| **Memory** | 8 GB RAM | 16 GB RAM |
| **Graphics** | GTX 1060 / RX 580 | RTX 2070 / RX 5700 XT |
| **Storage** | 20 GB available space | 50+ GB on an SSD |

:::warning Storage Space
s&box acts as a platform. Every new game you join downloads its unique assets (maps, models, code). If you play many different games, your installation folder will grow. An SSD is highly recommended to speed up these loading times.
:::

## Troubleshooting Installation

* **Game Won't Launch:** Try verifying the integrity of game files through Steam (Right-click s&box -> Properties -> Local Files -> Verify integrity).
* **Missing Executable:** Your antivirus might have falsely flagged an engine component. Add an exception for your steamapps/common/sbox folder and verify files.
